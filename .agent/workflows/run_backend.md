---
description: How to run the PHP Backend
---

1.  **Open XAMPP Control Panel**.
    - Search for "XAMPP Control Panel" in your Windows Start menu.

2.  **Start Apache and MySQL**.
    - Click the **Start** button next to **Apache**.
    - Click the **Start** button next to **MySQL**.
    - Ensure both turn green and show port numbers (usually 80 and 3306).

3.  **Verify Backend is Running**.
    - Open your browser and go to: [http://localhost/e-commerce/backend/api/products/read.php](http://localhost/e-commerce/backend/api/products/read.php)
    - You should see a JSON response with products.

> [!NOTE]
> No terminal command is needed to run the PHP backend if you are using XAMPP. It runs automatically as long as Apache is started.
